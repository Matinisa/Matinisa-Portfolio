# 🌍 Community Development NGO Website

## 📌 Project Overview

This project is a responsive website developed for a Non-Governmental Organization (NGO) focused on community development. The platform is designed to inform users about the NGO’s mission, showcase ongoing projects, and allow individuals to get involved through volunteering and donations.

The goal of this project is to simulate a real-world NGO website while applying front-end development skills and basic user interaction handling.

---

## 🎯 Objectives

- Build a multi-page, user-friendly website
- Provide information about community initiatives
- Enable user interaction through forms (volunteering & donations)
- Practice structuring real-world web applications

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout  
- **JavaScript** – Form validation and interactivity  

---

## ✨ Features

### 🏠 Home Page
- Overview of the NGO
- Mission and vision statement
- Call-to-action sections

### 📂 Projects Page
- Displays ongoing and completed community projects
- Highlights impact and involvement opportunities

### 🙋 Volunteer Sign-Up Form
- Users can register as volunteers
- Input validation using JavaScript
- Collects user details (name, email, interests, etc.)

### 💰 Donation Page
- Allows users to support the NGO financially
- Simple interface for donation input

### 📱 Responsive Design
- Optimized for desktop and mobile viewing
